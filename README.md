## Requirements

```bash
pip install opencv-python
pip install opencv-contrib-python==3.4.2.17
pip install argparse
pip install numpy
git clone https://github.com/mradul2/vision-tasks
```

`opencv-contrib-python` is installed for using SIFT detector.
